
## A utility function to do inorder tree traversal 
```python
def inorderID(root): 
    if root: 
        inorderID(root.left) 
        print(root.val) 
        inorderID(root.right)
        
def inorderName(root):
    if root:
        inorderName(root.left)
        print(root.employeeName)
        inorderName(root.right)
```
