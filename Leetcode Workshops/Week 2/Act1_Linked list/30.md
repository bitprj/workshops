## Step2:
### Another way: add a new node to our *film* Linked List is by using the *insert()* function. 

Assume we got extra appointments for auditions up until lunchtime.

```python 
film.insert(1030)
film.insert(1100)
film.insert(1130)
film.insert(1200)
film.insert(1230)
```
