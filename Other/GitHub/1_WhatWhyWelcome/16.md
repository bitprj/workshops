Title: Git/GitHub Terms (3)

Text: 

* **Fork**
  * make a copy of someone else repository
* **Pull request**
  * request to add your changes into a forked repository 
* **Clone**
  * copy a repository onto your local computer
* **Download Zip**
  * download the content of a repository
