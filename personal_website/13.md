# Adding Text to the Body
-  Add your name in a heading tag,
-  Add your description in a paragraph tag, in between the opening (`<body>`) and closing (`</body>`) tags. 
```html
<!DOCTYPE html>
<html>
  <head>
  </head>
  <body>
    <h1>Prophet Orpheus</h1>
    <p>Coder Dino
    Will code for food</p>
  </body>
</html>
```
